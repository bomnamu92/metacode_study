- project idea: 영화 추천 챗봇
  .상황/조건을 제시하거면 그 상황/조건에 맞는 영화를 추천해준다. 
  .영화를 제시하고 비슷한 영화를 추천해 달라고 하면 추천해 준다. 
- 데이터 셋 
  .MovieLens: 1M ratings, 6000users on 4000 movies
  .TMDb: API 제공
- 유즈케이스
  1) 자연어로 영화 추천 요청
   .사용자: 주말 저녁에 아이들과 보기 좋은 영화 추천해 줘.
   .시스템: 입력 문장에서 핵심 조건 추출
   .시스템: 영화 DB에서 관련 영화 검색
   .시스템: 추천 결과 정리, 추천 영화 목록 및 추천이유를 보여줌. 
  2) 조건 기반으로 영화 추천 요청
   .사용자: 2시간 이내의 아이들도 볼 수 있는 애니메이션 영화 추천해줘. 
   .시스템: 입력 문장에서 조건 추출
   .시스템: 조건에 맞는 영화 검색
   .시스템: 우선순위에 따라 정렬하여 영화 목록 보여줌.
- 추가 기능: 
   .영화 검색/추천
   .취향 분석
   .추천 이유 설명
- 구현안
  w1) 주제 확정, 데이터 수집, 데이터 구조 정리
  w2) 기본 검색/필터 기능 구현, 장르/평점 연도 기반 추천
  w3) 유사 영화 검새 구현
  w4) LangChain 자연어 질의 처리, 추천 이유 생성
  w5) 입/출력 정리
  w6) 테스트
